# Face_mask_detection_using_transfer_learning_with_MobileNet

1. To use the code, download the code.ipynb
2. Run through each cell after importing the necessary libraries.
3. Download the dataset from the link in Dataset file
4. You can skip the pickle cells to skip exporting of preprocessed image in pickle format and directly move to training your model
5. Last cell contains code for predicting from live webcam feed.


** To use my trained model, download it from the "Model" folder with the link and load it into your notebook
